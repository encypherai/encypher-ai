# StreamingMetadataEncoder

This page documents a legacy `StreamingMetadataEncoder` API that is no longer part of the `encypher-ai` public package.

For streaming LLM output, use:

- `StreamingHandler` (streaming embedding)
- `UnicodeMetadata.verify_metadata` (verification)

See:

- [`StreamingHandler`](../streaming/handlers.md)
- [`UnicodeMetadata`](./unicode_metadata.md)
