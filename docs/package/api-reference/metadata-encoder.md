# MetadataEncoder

This page documents a legacy `MetadataEncoder` API that is no longer part of the `encypher-ai` public package.

If you are looking for metadata embedding/extraction/verification, use:

- `UnicodeMetadata` (core embedding + verification)
- `StreamingHandler` (streaming LLM output)

See:

- [`UnicodeMetadata`](unicode_metadata.md)
- [`StreamingHandler`](../streaming/handlers.md)
