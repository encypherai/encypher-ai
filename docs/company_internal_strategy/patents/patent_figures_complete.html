Dual-Binding Manifest Construction Sequence

Step 1
Hash the Text Content (1010)
H_text = SHA256(normalize(text))
Step 2
Create Preliminary Manifest with Placeholder (1020)
preliminary_manifest = {
  content_hash: H_text,
  merkle_root: H_ABCDEFGH,
  assertions: [...],
  self_hash: PLACEHOLDER
}
Step 3
Hash the Preliminary Manifest (1030)
H_preliminary = SHA256(preliminary_manifest)
Step 4
Replace Placeholder → Create Final Manifest (1040)
final_manifest = {
  content_hash: H_text,
  merkle_root: H_ABCDEFGH,
  assertions: [...],
  self_hash: H_preliminary
}
Step 5
Digitally Sign the Final Manifest (1050)
signed_manifest = sign(final_manifest, private_key)
Step 6
Embed Signed Manifest in Text (1060)
authenticated_text = text + encode_unicode(signed_manifest)

Dual Binding: The manifest binds to both (1) the text content via H_text, and (2) itself via H_preliminary. This ensures any modification to either the text OR the manifest after signing will be detected during verification.
