# Questions

This directory contains questions from AI agents that require USER clarification.

## Naming Convention

```
TEAM_XXX_<topic>.md
```

## Template

```markdown
# Question: <Topic>

**Team**: TEAM_XXX  
**Date**: YYYY-MM-DD  
**Status**: open | resolved

## Context
Brief context about what the agent was working on.

## Question
The specific question requiring clarification.

## Options Considered
1. Option A: description
2. Option B: description

## Resolution
(Filled in after USER responds)
```

## Rules
1. Create question file BEFORE guessing on major decisions
2. Be specific about what you need clarified
3. Provide options you've considered
4. Mark as resolved when answered
